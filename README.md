## Usage

Clone the repo and install dependencies:

```bash
npm install
```

### Common Commands

| Command                   | Description                        |
|---------------------------|------------------------------------|
| `npm run dev`             | Local dev server                   |
| `npm run build`           | Build for production               |
| `npm run preview`         | Preview production build           |
| `npm run lint`            | Run ESLint                         |
